Detailed Description

The TLMHub singleton manages TLMMyos.

Method Documentation

- (NSArray *) myoDevices

A snapshot of the list of TLMMyos currently connected to TLMHub.

Returns
NSArray containing the connected TLMMyos.
- (void) pairWithAdjacent

Calling this method enables pairWithAdjacent mode. When a Myo touches the iOS device, the TLMHub will connect to it. If there are too many Myos connected, the first Myo in the myoDevices array is disconnected. Will continue to scan until stopPairWithAdjacent is called.

See Also
- stopPairWithAdjacent
- (void) pairWithAny

Searches for any TLMMyo and connects to it. Once connected, the TLMMyo will be paired, and TLMHub will try to connect to it as often as it can. Once this method results in the pairing of a Myo, it needs to be called again in order to pair with another Myo.
